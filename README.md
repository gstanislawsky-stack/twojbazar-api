# TwojBazar

Główny projekt znajduje się w folderze:

`twojbazar.com/`

Najważniejsze części:

- frontend web: `twojbazar.com/index.html`
- formularz dodawania: `twojbazar.com/add-listing.html`
- mobilna wersja web: `twojbazar.com/app/`
- backend Node.js: `twojbazar.com/server.js`
- aplikacja Flutter WebView: `twojbazar.com/flutter_app/`

Repo zostało uporządkowane tak, aby pliki projektu były trzymane w jednym miejscu.
